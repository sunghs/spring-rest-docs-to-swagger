# spring-rest-docs-to-swagger

Spring Rest Docs 기반의 테스트 코드를 Swagger에서 사용할 수 있도록 하는 예제 프로젝트입니다.

# Environment
- spring boot 2.5.4
- gradle 7.1.1
- Java 11

reference : https://github.com/ePages-de/restdocs-api-spec
